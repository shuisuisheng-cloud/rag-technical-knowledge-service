## 常见错误

- DATA 没有上拉；
- GPIO 一直保持输出模式；
- 使用推挽输出与 DHT11 抢占总线；
- 使用毫秒延时处理微秒时序；
- 等待电平没有超时；
- 将一次响应成功误认为完整数据读取成功；
- 引脚电平有效关系理解错误。

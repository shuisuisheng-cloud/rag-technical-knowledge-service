# LeetCode Day 8 快乐数

## 题目类型

数学、哈希集合、循环检测。

## 题目目标

对于一个正整数，不断计算它各位数字的平方和。

如果计算过程最终得到 1，则这个数字是快乐数；如果计算过程进入重复循环，始终不能到达 1，则不是快乐数。

例如：

`19 → 82 → 68 → 100 → 1`

因此 19 是快乐数。

## 核心函数

### `get_next_number(number)`

该函数只负责完成一轮各位数字平方和计算。

输入：

一个正整数，例如 19。

输出：

该整数各位数字平方和，例如 82。

处理流程：

1. 将整数转换为字符串；
2. 遍历每一个数字字符；
3. 使用 `int()` 将字符转换为整数；
4. 计算平方并累加；
5. 返回本轮计算结果。

函数职责单一，使主程序只需要负责判断是否到达 1，以及是否出现循环。
